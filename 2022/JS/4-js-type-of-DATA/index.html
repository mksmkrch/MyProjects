Типы данных в JavaScript

  • JavaScript - динамически типизированный, т.е. тип данных переменной меняется в момент присвоения либо смены её значения
  • Оператор typeof возвращает строку, содержащую информацию о типе операнда
  • Тип undefined вернет объявленная неопределенная (без присвоенного значения) переменная . Возвращает тип данных - undefined, значение - indefined
  • Тип null возвращает объявленная определенная переменная, её значением будет - "ничего". Но вот тип у такой переменной - object. Возвращает тип данных - object, значение - null
  • Тип boolean - булевый или логический тип. Возвращает тип данных boolean, значение false\true.
  • Тип number числовой тип данных, представляет значения в виде целых чисел и чисел с плавающей точкой. Возвращает тип данных number. Значения - число. При делении на ноль (let getInfinity = 25 / 0;) - Infinity/-Infinity. При вычислительной ошибке - NaN. Например при let getNan = 'abc' / 10
  • Тип bigint. В JS тип данных number не может содержать числа больше, чем 9007199254740991, или меньше, чем -9007199254740991.
  • Тип string возвращает переменная определение в кавычках, любых из трёх видов.
  • Тип object используется для хранения коллекций различных значений и более сложных сущностей.
  • Тип symbol
  • Тип function тот же object
